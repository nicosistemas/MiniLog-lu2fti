# MiniLog

Simple logger developed by LU2FTI.

`pip install Flask`

`python3 app.py`


Feature:

- Registry ID Call, frecuency, mode
- Export to Adif format (.adi)
- Posibility to change ID
- Database on simple csv file
- UTC Hour
- Save the idcall record with the Guardar button or Enter
- Focus in ID Call
